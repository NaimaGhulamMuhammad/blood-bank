# blood-bank-App
**User can register as a donar or a user and then complete the profile by providing blood information and can donate or search accordingly**
<br />**TECH STACK:
------------------------------------

* ES+6
* Material-ui
* ReactJs
* Redux, Redux Thunk
* Firebase Auth (Authentication)
* Firebase Firestore (Database)
* Firebase (Hosting)
